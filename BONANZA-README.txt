1. 本パッケージについて

  アーカイブ bonanza_v4.1.3.zip により配布されます．このアーカイブを展
  開すると以下のファイルとフォルダ（ディレクトリ）が生成されます．

    readme.txt        本ドキュメント
    winbin/           Windows 実行形式
    src/sikou_dll/    外部プログラムからの Windows 実行形式の利用
    src/book/         序盤データベースのファイル構造
    src/bonanza/      Bonanza ソースファイル一式

  このパッケージをダウンロードしてくださった大抵の方々にとって必要なファ
  イルは，フォルダ winbin/ の下にあります．ここには Bonanza の Windows
  実行形式に関するファイル一式があり，winbin/winbin.txt に簡単な説明が
  記述されています．

  他のフォルダは，将棋プログラムを作成されている方々にとって有用なもの
  です．src/sikou_dll/sikou_dll.txt, src/book/book.txt,
  src/bonanza/bonanza.txt をお読み下さい．
